Je dáno pole s kladnými čísly a číslo N. Vaším úkolem je najít N-tou mocninu prvku pole s indexem N. Pokud je N mimo rozsah pole, potom vraťte číslo -1. Nezapomeňte, že první prvek pole má vždy index 0.

Pojďme se podívat na pár příkladů:
- pole = [1, 2, 3, 4] a N = 2, potom je výsledkem 32 == 9;
- pole = [1, 2, 3] a N = 3, ale N je mimo rozsah pole, a proto je výsledkem -1.

Vstup: Dva argumenty. Pole jako seznam integerů a číslo jako integer.

Výstup: Výsledek jako integer.

Příklad: 

index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1

K čemu to je: Tato mise Vás naučí, jak používat základní pole a indexy společně s jednoduchou matematikou.

Předpoklady: 0 < len(array) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in array)
