Σου δίνεται ένας πίνακας με θετικούς αριθμούς και ένας αριθμός Ν. Πρέπει να βρείς τη Ν-ιοστή δύναμη του στοιχείου του πίνακα που βρίσκεται στη θέση Ν. Αν το Ν είναι έξω από τα όρια του πίνακα τότε θα επιστρέφεται ο αριθμός -1. Μην ξεχνάς ότι το πρώτο στοιχείο του πίνακα βρίσκεται στη θέση 0.

Ας δούμε μερικά παραδείγματα:
- array = [1, 2, 3, 4] και N = 2, τότε το αποτέλεσμα είναι 32 == 9;
- array = [1, 2, 3] και N = 3, το Ν όμως είναι έξω από τα όρια του πίνακα, οπότε το αποτέλεσμα είναι -1.

Είσοδος: Δύο παράμετροι. Ένας πίνακας με μορφή λίστας ακεραίων και ένας ακόμα ακέραιος αριθμός.

Έξοδος: Το αποτέλεσμα με μορφή ακεραίου.

Παράδειγμα: 

index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1

Πώς χρησιμοποιείται: Αυτή η αποστολή σου διδάσκει πώς να χρησιμοποιείς πίνακας και δείκτες σε συνδυασμό με απλά μαθηματικά.

Προϋποθέσεις: 0 < len(array) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in array)
